# docs/

Project documentation for human readers. Design documents, guides, references, and any prose that doesn't belong in code comments or CLAUDE.md files.

## Conventions

- Write for humans, not for Claude (Claude-facing docs go in per-directory `CLAUDE.md` files)
- Use descriptive filenames: `design_[topic].md`, `guide_[topic].md`, `reference_[topic].md`
- Keep documents self-contained where possible
