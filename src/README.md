# src/

Source code for the project. Contains Python modules organized by responsibility.

## Structure

```
src/
  [package_name]/     # Main package directory
    __init__.py
    ...               # Modules organized by domain
```

## Conventions

- All importable code lives here
- Follow the naming and style conventions defined in the root `CLAUDE.md`
- Module-level docstrings are required for new files
